# Thegioididong Landing Page Clone

A static clone of Thegioididong homepage built with pure HTML, CSS, and JavaScript.

## Features

- Pure HTML, CSS, JavaScript (no libraries)
- Responsive layout (desktop, tablet, mobile)
- Landing page only (homepage)
- Clean folder structure

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
